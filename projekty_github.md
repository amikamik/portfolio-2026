# Podsumowanie Projektów GitHub

## Narzędzia i Automatyzacja (Python/AI)
1. **Analizator-Ogloszen-OLX / olx-kategoryzator / olx-category-tool**
   Zaawansowane skrypty służące do scrapowania, analizy i automatycznej kategoryzacji ogłoszeń z serwisu OLX, często wspierane przez algorytmy AI. Służą do monitorowania rynku i porządkowania dużych wolumenów danych ogłoszeniowych.

2. **instant-mesh-api**
   API służące do obsługi technologii Instant Mesh (generowanie modeli 3D). Pozwala na programistyczny dostęp do generowania lub przetwarzania obiektów trójwymiarowych.

3. **analiza_danych / analiza_danych_python**
   Projekty skupione na Data Science – czyszczenie, przetwarzanie i wizualizacja danych statystycznych. Pokazują warsztat analityczny i umiejętność pracy z bibliotekami takimi jak Pandas czy Matplotlib.

## E-commerce i Strony Produktowe
4. **sklep-kettle**
   Projekt sklepu internetowego lub strony sprzedażowej dedykowanej sprzętowi sportowemu (odważniki kettlebell).

5. **sklep-kominiarki**
   Witryna e-commerce lub landing page skupiony na sprzedaży odzieży specjalistycznej (kominiarek).

6. **szyby_kominkowe**
   Strona produktowa lub sklep dla niszowej branży akcesoriów kominkowych (szyby żaroodporne).

## Web Development i Projekty Osobiste
7. **afiliacja**
   Projekt związany z marketingiem partnerskim – prawdopodobnie system do zarządzania linkami lub strona typu "best picks" kierująca ruch do ofert zewnętrznych.

8. **uslugi-przez-internet**
   Serwis internetowy prezentujący katalog usług możliwych do realizacji zdalnej, pełniący funkcję platformy ogłoszeniowej lub wizytówki.

9. **projekt-frontend / strona_glowna / amikamik.github.io**
   Projekty frontendowe stanowiące Twoją osobistą wizytówkę w sieci oraz poligon doświadczalny dla technologii webowych (HTML, CSS, JavaScript).
