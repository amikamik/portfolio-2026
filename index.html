System Automatyzacji Sprzedaży OLX (AI)

Zaawansowana platforma integrująca zewnętrzne feedy produktowe (XML) z API OLX. System wykorzystuje modele LLM (OpenAI/Gemini) do semantycznej analizy produktów, automatycznie mapując je do drzewa kategorii OLX i uzupełniając atrybuty.

Rozwiązanie automatyzuje cały lejek sprzedażowy: od pobrania danych, przez kalkulację marż i zysków, aż po weryfikację jakości ogłoszenia i publikację przez Oficjalne Partner API.

Python & XML OpenAI / Gemini API OLX Partner API Automatyzacja E-commerce

Automated Data Analysis API

Nowoczesny mikroserwis backendowy stworzony w FastAPI, który democratyzuje analizę danych. Użytkownik przesyła surowy plik CSV i natychmiast otrzymuje kompleksowy raport statystyczny.

System automatycznie wykonuje czyszczenie danych (Pandas), generuje profilowanie (YData) i udostępnia wyniki przez REST API. Idealne narzędzie do szybkiego prototypowania modeli ML.
