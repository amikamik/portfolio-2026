Android Mobile App

Radar Opadów z Kompasem Kierunkowym (Android)

Zaawansowana aplikacja mobilna z predykcją opadów i kompasem kierunkowym. System wykorzystuje czujniki orientacji telefonu (magnetometr, akcelerometr) do wizualizacji kierunku nadchodzących opadów atmosferycznych.

Aplikacja analizuje dane z Open-Meteo API (minutely_15 dla krótkiego zasięgu, hourly dla długoterminowych prognoz) i RainViewer radar, wyświetlając dokładny czas nadejścia opadu (ETA) oraz jego intensywność. Kompas automatycznie dostosowuje się do orientacji urządzenia, wskazując rzeczywisty kierunek geograficzny nadchodzących frontów atmosferycznych. Dodatkowo monitoruje siłę sygnału komórkowego (dBm) dla lepszej świadomości łączności podczas złych warunków pogodowych.

Kotlin Android SDK & Jetpack Compose Czujniki: Magnetometr/Akcelerometr Open-Meteo API RainViewer Radar Location Services & TelephonyManager
